# MMDB

- [https://ip.circl.lu/](https://ip.circl.lu/) / [adulau/mmdb-server](https://github.com/adulau/mmdb-server)

This enricher uses public MMDB API to enrich an IP artifact.

```yaml
enricher: mmdb
```

This enricher can add the following components:

- Geolocation
- Autonomous System

## Supported Artifacts

- IP address
